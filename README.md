# Janice-Chan-Linear-Algebra-End-of-Semester-2-Overview

My project is an exploration of applying linear algebraic techniques to recommendation models. Rather than focusing on research, I chose to construct three programs in python that look at how three different approaches could be used for a recommendation model. The programs are situated in the context of a dating app that recommends the user vectors that represent humans.

The project consists of the three programs (decorated with comments designed to help the reader follow along with the code) and a reflection that details comparisons between approaches, short-comings of each approach, and challenges I encountered while coding.

You can find my code, stored in this GitHub repository, and I encourage you to test out each program—think of a preference/type that one human could have for another and see if the program can recommend humans that capture that preference. Also check out the reflection for a deeper dive into each approach.
